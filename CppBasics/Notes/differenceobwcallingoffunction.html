Difference Between Call by Value, Call by Address, and Call by Reference

FeatureCall by ValueCall by AddressCall by Reference
Argument PassedCopy of the actual valueAddress (pointer) of the variableReference (alias) to the variable
Original Value ChangedNoYesYes
EfficiencyLess efficient for large dataEfficient for large dataEfficient and easy to use
SyntaxBasic, no extra syntaxUses pointers and*,&operatorsUses reference&
Common Use CasesSimple data types, local variablesDynamic memory, low-level programmingWhen changes to original data are needed
